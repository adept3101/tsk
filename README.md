# Инструкция

1. Клонировать репозиторий:
```
git clone https://github.com/adept/tsk
```

2. Перейти в директорию ``lib`` и собрать библиотеку:
```
cd lib
make
```
3. Перейти в директорию ``1`` и собрать первое приложение:
```
cd 1 
mkdir build
make
```
4. Перейти в директорию ``2`` и собрать второе приложение:
```
cd 2
mkdir build
make
```

## Как пользоваться?

Исполняемый файл находится в папке ``build`` запускать командой: 
```
./main
```
